Ausca

Ausca is a collection of automation utilities and bots written in Ruby.

Installation

Add this line to your application's Gemfile:

gem 'ausca'

And then execute:

$ bundle

Or install it yourself as:

$ gem install ausca

Usage

Check the examples directory for functional examples.
